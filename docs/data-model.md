- 테이블명은 직관적으로 지어야 한다.
- 데이터는 정확해야 한다.

- 테이블명은 복수로, 컬럼명은 단수로.
- 약어는 지양해야 한다.
- table명을 column명에 똑같이 쓰는 것 자제
    - pk로 사용하는 id는 테이블명_id로 쓰지말고 그냥 id로
    - ex) book 테이블에 book_content (x)  / content (o)
