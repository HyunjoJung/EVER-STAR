# 기능/요구사항 정리

## 핵심 서비스 컨셉
- 펫로스 증후군 극복/일상 회복을 돕는 웹 서비스.
- 지구별(사용자 공간)과 영원별(반려동물 추모 공간)로 나눠 무지개 다리 컨셉 적용.

## 주요 기능
- **49일 퀘스트**: 무지개 7단계+일상 회복 단계로 일일 퀘스트 제공, 감정 정리 유도. 설정한 시간에 도착.
- **편지 쓰기 & GPT 답장**: 반려동물에게 편지를 작성하면 다음 날 GPT 기반 답장을 제공. 퀘스트 내 “깜짝 답변”도 제공.
- **감정 분석 시각화**: 네이버 감정분석 결과를 날씨/단계로 표현, 추세 확인.
- **메모리얼북**: 49일 퀘스트 완료 시 열람 가능한 엔드 콘텐츠. 답변 기록/감정 분석 결과 포함, PDF 저장 가능.
- **응원/커뮤니티**: 지인이 응원 메시지를 남길 수 있는 안전한 공간(욕설 필터링).
- **랜덤 탐험**: 다른 영원별(추모별)로 랜덤 이동하여 위로/공감 제공.
- **마이페이지**: 프로필/반려동물 선택 및 설정.
- **화상 통화(OpenVidu)**: 가족/지인과 화상 연결(음소거/비디오/스피커 토글, 초대 기능).
- **알림**: 편지 도착/퀘스트 도착·완료 알림. 실서비스는 FCM, 로컬/E2E는 비활성화.

## 기술적 요구사항 (요약)
- 프론트: React/TS, PWA, MSW로 모킹, Storybook(아토믹 디자인), OpenVidu 연동.
- 백엔드: Spring Boot, JPA, MySQL. SSE/스케줄러, Kafka 언급, OAuth(Kakao/Google), FCM.
- AI: GPT API(편지/답변), 네이버 감정분석 API.

# sms 인증 테스트

상세 페이지: 회원가입
epic: 회원관리
구현: Yes
기능 상세(버튼,하위컴포넌트): 사용자 전화번호 기반 인증
기능(컴포넌트): 전화번호 인증
디자인: No
설명: 유효한 전화번호인지 인증
우선순위: 0순위-필수

- **문자전송 성공 테스트 (`sendSms`)**:
    - 정상적인 전화번호를 가진 요청을 보내고, `sendSms` 메소드가 호출된 후 `SmsCertificationUtil`의 `sendSms`와 `SmsCertificationRepository`의 `createSmsCertification` 메소드가 호출되는지 확인합니다.
- **문자전송: 유효하지 않은 전화번호 형식 테스트 (`sendSms_invalidPhoneFormat`)**:
    - 잘못된 전화번호 형식이 주어진 경우 `sendSms` 메소드가 `IllegalArgumentException`을 던지는지 확인합니다.
- **문자인증 성공 테스트 (`verifySms`)**:
    - 올바른 인증번호와 전화번호를 가진 요청을 보내고, 인증이 성공적으로 이루어지면 `SmsCertificationRepository`의 `deleteSmsCertification` 메소드가 호출되는지 확인합니다.
- **문자인증 실패 테스트 (`verifySms_fail`)**:
    - 인증번호가 잘못된 경우 `verifySms` 메소드가 `RuntimeException`을 던지고, `deleteSmsCertification` 메소드가 호출되지 않는지 확인합니다.
- **문자인증: 유효하지 않은 전화번호 형식 테스트 (`verifySms_invalidPhoneFormat`)**:
    - 잘못된 전화번호 형식이 주어진 경우 `verifySms` 메소드가 `IllegalArgumentException`을 던지는지 확인합니다.
- **문자인증: 유효하지 않은 전화번호 null 테스트 (`verifySms_nullPhone`)**:
    - 전화번호가 `null`인 경우 `verifySms` 메소드가 `IllegalArgumentException`을 던지는지 확인합니다.

## 메모리얼북/일기 설계

### 메모리얼북에 들어가는 정보

```java
/*
	반려동물 객체
	id, 보호자 id, 이름, 나이, 기일, 종류, 성별, 관계, 프로필 사진,
	설명, 성격들
*/
class Pet {
	id, userId, name, age, memorialDate, species, gender, relationship,
	profileImage, introduction, List<Personality>
}

/*
	감정 분석 결과 객체
	id, 결과, 주차별 분석 결과 (Naver 감정분석 AI API 사용)
*/
class SentimentAnalysis {
	id, result, analysis_week1 ~ analysis_week7
}

/*
	질문 객체
	id, 내용, 유형
*/
class Quest {
	id, content, type
}

/*
	답변 객체
	반려동물 id, 질문 id, 내용, 이미지url, 비디오url, 유형
	질문 유형에 따라 내용, 이미지, 비디오 필드에 값이 들어감
*/
class QuestAnswer {
	petId, questId, content, imageUrl, type
}

/*
	AI 답변 객체
	반려동물 id, 질문 id, 내용, 이미지url
	질문 유형에 따라 내용, 이미지 필드에 값이 들어감
*/
class AiAnswer {
	petId, questId, content, imageUrl
}

/*
	메모리얼북 객체
	id, 반려동물 id, 우울진단심리검사 결과, 공개여부, 활성화여부
*/
class MemorialBook {
	id, petId, psychologicalTestResult, isOpen, isActive
}

/*
	일기 객체
	id, 메모리얼북 id, 제목, 내용, 이미지, 생성시간
*/
class Diary {
	id, memorialBookId, title, content, imageUrl, createdDate
}
```

# 1. 목표

- 아래 응답 예시 결과를 하나의 변수로 만들고 변수의 값이  0과 1 사이 값을 갖도록 변환하고자 함. 따라서 하나의 변수 값을 통해 부정부터 긍정을 나타낼 수있도록 변환함.
- 변환 된 값을 아래 6분위(미정)로 나누어 날씨 이모티콘으로 표현
- 7주라서 6-7가지 지표 필요할 것 같다고 생각함
    
    
    | **1** | **먹구름 천둥번개(매우부정)** | **0 ~ 0.166667** | 천둥(매우부정) | 0-0.2 |
    | --- | --- | --- | --- | --- |
    | **2** | **비(부정)** | **0.166667 ~ 0.333333** | 비(부정) | 0.2-0.4 |
    | **3** | **구름 많음(부정중립)** |  **0.333333 ~ 0.5** | 바람(중립) | 0.4-0.6 |
    | **4** | **구름 조금(긍정중립)** | **0.5 ~ 0.666667** | 구름(긍정) | 0.6-0.8 |
    | **5** | **맑음(긍정)** | **0.666667 ~ 0.833333** | 맑음(매우긍정) | 0.8-1.0 |

# 2. 분석 결과 가공

### 변환 과정

1. 주어진 감정 분석 결과를 하나의 숫자 값으로 변환하여 부정에서 긍정을 나타내도록 만들기 위해, 주어진 감정 신뢰도를 하나의 스칼라 값으로 매핑
2. 부정적 감정을 0, 중립적 감정을 0.5, 긍정적 감정을 1로 매핑
3. 각 감정의 신뢰도를 가중치로 사용하여 하나의 값을 계산
4. 위 방법을 통해, 주어진 신뢰도를 하나의 값으로 결합

<aside>
🌼 **sentiment Score 식 산출 :** 0 ~ 1 ( 부정 ~ 긍정)

$$
sentimentScore=negativeConfidence×0+neutralConfidence×0.5+positiveConfidence×1
$$

</aside>

### 분석 예시 1 (네이버 api 예시-부정 case)

- Neutral: 0.14525136640572725
- Positive: 0.00186876227013191
- Negative: 0.8528798713241407

$$
sentimentScore=(0.8528798713241407×0)+(0.14525136640572725×0.5)+(0.00186876227013191×1)
$$

이 식을 계산하면:

$$
sentimentScore=0+0.07262568320286362+0.00186876227013191=0.07449444547299553
$$

따라서, 변환된 sentiment score는 0.0745로, 이는 부정적인 감정에 더 가까움

---

### 분석 예시 2(중립 case)

1. **Neutral: 0.5, Positive: 0.25, Negative: 0.25**

$$
sentimentScore=0.5×0.5+0.25×1+0.25×0=0.25+0.25=0.5
$$

1. **Neutral: 0.3333333333333, Positive: 0.3333333333333, Negative: 0.3333333333333**

$$
sentimentScore=0.3333333333333×0.5+0.3333333333333×1+0.3333333333333×0=0.16666666666665+0.3333333333333=0.49999999999995
$$

1. **Neutral: 0.8, Positive: 0.1, Negative: 0.1**

$$
sentimentScore=0.8×0.5+0.1×1+0.1×0=0.4+0.1=0.5
$$

각 경우에 대한 sentiment score는 모두 0.5임. 이는 세 가지 사례에서 감정의 균형이 동일하게 중립적임을 나타냄

# 4. 코드

```java
public class SentimentAnalysis {

    public static void main(String[] args) {
        // 예시 감정 분석 결과 -> 
        double neutral = 0.14525136640572725;
        double positive = 0.00186876227013191;
        double negative = 0.8528798713241407;

        double sentimentScore = calculateSentimentScore(neutral, positive, negative);
        String weatherEmoji = getWeatherEmoji(sentimentScore);

        System.out.printf("Sentiment Score: %.4f\n", sentimentScore);
        System.out.println("Weather Emoji: " + weatherEmoji);
    }

    /**
     * 감정 분석 결과를 기반으로 하나의 숫자 값을 계산 함수
     * @param neutral 중립 신뢰도
     * @param positive 긍정 신뢰도
     * @param negative 부정 신뢰도
     * @return 0에서 1 사이의 감정 점수
     */
    public static double calculateSentimentScore(double neutral, double positive, double negative) {
        // 부정, 중립, 긍정 신뢰도를 0, 0.5, 1로 매핑
        double negativeWeight = 0;
        double neutralWeight = 0.5;
        double positiveWeight = 1;

        // 각 감정의 신뢰도를 가중치로 사용하여 하나의 값을 계산
        return (negative * negativeWeight + neutral * neutralWeight + positive * positiveWeight);
    }

    /**
     * 감정 점수를 기반으로 날씨 이모티콘을 반환 함수 => 프론트와 협의 필요 ! 
     * @param sentimentScore 0에서 1 사이의 감정 점수
     * @return 날씨 이모티콘
     */
    public static String getWeatherEmoji(double sentimentScore) {
        if (sentimentScore < 0.166667) {
            return "🌩️ 먹구름 천둥번개";
        } else if (sentimentScore < 0.333333) {
            return "🌧️ 비";
        } else if (sentimentScore < 0.5) {
            return "☁️ 구름 많음";
        } else if (sentimentScore < 0.666667) {
            return "⛅ 구름 조금";
        } else if (sentimentScore < 0.833333) {
            return "☀️ 맑음";
        } else {
            return "🌈 무지개";
        }
    }
}
```

# 유빈 정리

**반려동물을 상실한 성인여성의 애도중심 미술치료 개인 사례연구**

![Untitled](images/image_0048.png)/%EC%9C%A0%EB%B9%88%20%EC%A0%95%EB%A6%AC/Untitled.png)

**연구 기간 및 방법**

반려동물을 상실한지 2개월된 38세 기혼여성을 연구대상으로 선정하여 Kübler-Ross의 애도과정 5단계를 바탕으로 구성한 애도중심 미술치료 프로그램을 2019년 4월 4일부터 2020년 1월 30일까지 주 1~2회, 한 회기 당 70분 내외씩 총 34회기 진행

**연구 결과**

- 첫째, 반려동물 애도의 비애/슬픔, 트라우마, 죄책감 영역 점수가 감소되었고, 성인여성의 우울 점수가 크게 감소
- 둘째, 반려동물을 상실한 성인여성의 애도중심 미술치료 회기 및 단계에 나타난 애도와 우울의 변화를 살펴본 결과, 미술치료 작업과정과 결과물을 통해 반려동물 상실에 대한 현실 수용 및 새로운 삶의 의미를 통찰
- 우울감의 변화를 자각하고 일상생활을 회복하여 긍정적 변화가 나타났음

**요약**

펫로스 증후군 극복을 위한 애도중심 미술치료는 긍정적 효과가 있다.

**데스크 리서치 및 아이디에이션에 쓸만한 참고 내용**

- 반려동물 시장의 확대

‘2019년동물보호에 대한 국민의식 조사 보고서’에 따르면, 반려동물의 양육 가구는 2010년 17.4%에서 2019년 26.4%로 증가하였고, 전국 4가구 중 1가구는 반려동물을 기르는 수준이라 보고되었다(농림축산식품부, 2020).

농림축산식품부 (2020). 2019년 동물보호에 대한 국민의식 조사 보고서.

- 펫로스 증후군의 심각성

반려동물을 상실하게 되면 보통 2~3개월 정도의 애도 기간이 소요되며, 1년 이상 증상이 지속될 경우 치료가 필요할 수 있다. 증상이 심할 경우 복합 비애 또는 외상후 스트레스장애로 악화될 수 있다(모효정, 2015)

- 사회적 시선

반려동물의 상실에 따른 반려인의 비애와 슬픔이 인간과의 사별로 야기되는 슬픔과 유사함에도 불구하고, 반려동물의 상실로 인한 슬픔은 사회적으로 덜 인정받고 주변의 지지와 지원이 부족한 편이
다. 그래서 반려인들은 충분히 슬퍼하는 애도과정이 필요하지만, 주변 시선 때문에 마음껏 드러내 놓고 슬퍼하지 못하게 된다. 반려인의 슬픔을 위로해 주는 사회적 인식이 부족하기 때문에 이를 사회적으로 ‘인정받지 못한 비애(disenfranchised grief)’라고 한다(모효정, 2015).

모효정 (2015). 반려동물의 상실로 인한 슬픔, 펫로스(Pet Loss) 증후군의 증상과 대처. 인간·환경·미래, 15, 91-120.

- 애도의 필요성

Kast(2007)는 우리가 상실을 겪게 될 때나 무언가로부터 분리되어야만 할 때에는 항상 애도가 필요하다고 설명한다.

Kast, V. (2007). Trauern: Phasen und chancen des psychischen prozesses. Verlag Herder GmbH. (애도: 상실과 마주하고 상실과 더불어 살아가기. 채기화 역, 서울: 궁리, 2007).

애도과정이 성공적일 때 자아는 강화되고, 애도과정이 방해받을 때 자아는 정신 병리적 비탄에 빠지게 된다(장희은, 2015).

장희은 (2015). 배우자 사별 여성의 안심모형에 근거한 애도미술치료 경험 연구. 대구대학교 대
학원 박사학위논문

- 예술 치료의 장점

Irwin(1991)은 사별 가족 상담에서 예술을 사용하는 네 가지 장점에 대해 다음과 같이 설명한다. 첫째, 예술을 활용하는 것은 감정의 완화를 촉진하는데 도움을 주고 둘째, 애도자가 미처 의식하지 못하고 있던 갈등을 확인시켜준다. 셋째, 예술을 사용함으로써 당사자가 상실한 것이 무엇인지에 대한 자각을 깊게 해줄 수 있고 마지막으로 자신들이 애도과정 중 어디쯤 와 있는지 알려준다 (Worden, 2002/2009).

Worden. J. W. (2002). Grief counseling and grief therapy: A handbook for the mental health practitioner. New York: Springer Publishing Company. (유족의 사별슬픔 상담과 치료. 이범수 역, 서울: 도서 출판 해조음, 2009).

# 민경(의짱 동기)

1105_민경이 인터뷰  Q. 간단한 자기소개 부탁드립니다.

A.안녕하세요. 23살 전민경입니다.

q. 옛날에 키우던 반려동물을 소개해주실 수 있나요?

1. 제 반려동물은 고슴도치였구요. 이름은 도치고 남자 아이였습니다.

q. 지금 반려동물과 함께 살고 계신가요?

1. 지금은 자취를 하고 있어서 같이 살지는 못하는데 본가에 가면 강아지가 있습니다.

q. 제가 알기로는 반려동물 도치가 무지개 다리를 건넌 것으로 알고 있는데 그때의 상황을 설명해주실 수 있으신가요?

1. 제가 초등학교 때 도치를 키웠는데 도치가 엄청 작았어요. 다른 애들에 비해서 애가 엄청 작아서 귀엽고 잘 지냈는데 어느순간 애가 아프더라구요. 그래서 아빠가 병원도 자주 데려가고 그랬는데, 그 날 딱 가족끼리 외식을 하러 갔다가 집에 돌아왔는데 도치가 혼자 경련을 일으키면서 뻗어있는거에요. 그래서 깜짝 놀라서 다 같이 동물병원에 간 기억이 있어요.

q. 조금 조심스럽지만 반려동물과 이별을 겪었을 때 감정을 설명해주실 수 있으신가요?

1. 감정은 좀 어려서 구체적으로 기억이 나진 않는데 제가 계속 울었던 건 기억이 나요. 제가 어릴 때부터 강아지를 키우고 싶다는 생각이 많았는데 부모님이 강아지는 어려울 것 같다고 하셔서 고슴도치를 분양받아왔던 건데, 강아지처럼 제가 매일 손에 데리고 다니고 좋아해서 엄청 유대감이 있었던 것 같아요. 그래서 계속 울고 3일 정도는 밥도 안 먹고 그 초등학생 아기가, 그랬던 걸 부모님께 들었었습니다.

q. 그러면 보통 많은 반려인분들이 반려동물과 이별한 후, “있을 때 잘해줄 걸”와 같이 후회한다는 말을 많이 하세요. 그래서 민경씨도 도치가 떠난 후에 후회하셨던 경험이 있으신가요?

1. 음 약간 걔가 한 번씩 가시를 뽀족하게 세울 때가 있어요. 그러면 괜히 저는 아파서 도치한테 화내고 짜증내고... 그리고 이제 그 때는 어렸을 때니까 같이 있다가 다른 친구들 만나러 가거나... 되게 무관심해질 그런 시기가 있었어요. 한 한달정도? 그래서 그때 되게 엄마가 그럴거면 도치를 왜 키우냐고 말했던 적이 있는데, 그때가 되게 후회스러워요. 그때 더 많이 잘해줄걸... 그냥 예쁜 거, 리본 하나라도 달아줘볼걸... 사진도 찍을 걸... 사진이 남아있는 게 또 없더라고요. 사진 되게 많이 찍었던 걸로 기억나는데 하나도 없고... 그리고 내가 되게 관심이 없었나?이런 생각이 들어서 되게 아쉽고 미안했어요.

q. 사진같은 경우엔 드라이브와 같은 다른 곳에 미처 저장하지는 못한 건가요?

1. 네. 그때는 이제 2G폰 쓰고 이제 막 스마트폰이 나올 때라 드라이브 같은 것도 몰랐고 찍어둔 것이 없어진 거에요. 그래서 되게 아쉽고... 그래서 지금 반려동물 강아지 사진은 많이 찍으려고 하고 있습니다.

q. 반려동물과 함께 해서 행복했거나 기억에 남는 순간이 있으신가요?

1. 이거는 모든 반려동물을 키우시는 분들이 공감하실 수 있는데, 그 아기를 멀리 이렇게 놓아둬요. 그리고 앞에 가족들이 다 있고 이리로 오라고 막 말을 해요. 근데 도치가 매 순간 저한테만 왔어요.

q. 오 도치가 말을 알아들었나요? 

1. 그니깐 말을 못 알아듣거든요? 걔는 분명히. 근데 제가 이렇게 오라고 하면 걔는 와요 저한테. 뭐 때문에 온지는 저도 몰라요. 아무튼 그 순간이 되게 기억에 남아요.

q. 그러면 반려동물과 이별하기 전으로 간다면 반려동물과 함께 하고 싶은 것은 무엇인가요?

1. 전 그냥 얼굴이 보고싶어요. 왜냐하면 강아지처럼 산책을 갈 수 있는 것도 아니고 그러니깐... 그냥...

q. 그래도 지금은 어느정도 감정이 정리가 되신 것 같은데 도치랑 관련된 옛날 물건은 아직 가지고 계신가요?

1. 도치랑 관련된 거는 이제 다 버린 것 같아요. 없는 것 같아요 집에는.

q. 버리게 되신 이유가 보면은 생각이 나서라든지 그런 이유에서인가요?

1. 네. 그리고 저희도 집에 이제 그거를... 우리라고 하나? 그 집을, 플라스틱 통에서 키웠는데 엄청 한 2달 넘게? 한 3달 동안은 그 자리에 계속 놓아두었어요. 제가 계속 힘들어하고 엄마한테 치우지 말라고 하고 진짜 오래 그 자리에 있다가 이제 그래도 정리해야되지 않냐해서 그래서 치웠을 거에요.   

q. 제가 듣기로는 반려동물과 이별 후, 반려동물과 비슷한 인형을 만들어서 가지고 계셨다고 들었는데 그 에피소드에 대해 설명해주실 수 있으실까요?

1. 네. 제가 동물병원에 갔는데 제가 너무 어려서 그 때 충격을 되게 받은거에요. 동물이 막 경련을 일으키고 한 거에. 그래서 이제 아빠가 동물병원에 있고 엄마는 집에 동생과 저를 데리고 와서 있는데 아까 말한 케이지 안에 걔가 없다는 사실이 너무 허전하고 마음이 힘든 거에요. 그래서 그 때 왜 그랬는지 모르겠는데 재활용 통에 가서 요구르트 병을 최대한 비슷한 걸 막 찾은거에요 동생이랑 같이. 찾아서 막 이쑤시개 꼽아놓고 얘가 도치라고, 도치는 여기 다시 그대로 다시 올거라고. 그랬었어요. 병원에 가 있었는데 어차피 치료받고 오면은 다시 올거니까... 도치오면은 바꿔주자 이렇게 그랬었어요.

q. 그럼 반려동물과 이별한 후에 어떻게 감정을 정리하셨는지 여쭤봐도 괜찮을까요?

1. 음... 사실 감정 정리는 기억이 안나요. 그냥...  

q. 시간이 흐르면서 자연스럽게?

# 반려동물 장례 관련

많은 반려인들이 반려동물의 죽음으로 심리적, 정신적 고통을 겪고 있지만 정작 장례 절차나 방식에 대한 인식은 부족한 것으로 드러났다. 응답자 가운데 동물의 죽음을 경험한 이는 절반 가량이었지만, 실제로 장례를 치른 경험은 39.1%에 불과했다.

![](https://flexible.img.hani.co.kr/flexible/normal/800/533/imgdb/original/2022/0118/20220118500993.jpg)

반려동물 장례식장 ‘펫포레스트’의 추모공간. 사진 스튜디오 어댑터 윤동길

동물의 사체 처리 방법을 모르거나 잘못 알고 있는 비율도 40%에 달했다. 올바른 사체 처리법을 묻는 질문에 응답자의 22.3%는 ‘주변 산에 묻음’이라 답했고, ‘잘 모르겠다’는 응답은 17.8%였다.

**장례 정책의 필요성**

동물의 죽음을 경험하는 반려인들이 늘면서, 반려동물 사체 처리 방식이나 장례 정책에 대한 개선·보완의 필요성은 더 커지고 있다. 반려동물의 죽음을 마주했을 때 가장 절실했던 점이 무엇인지를 묻는 질문에 ‘장례식장 등 정보 부재’(33.3%)를 꼽은 응답자가 가장 많았다. 반려동물의 사체를 쓰레기로 규정하고 처리하는 조항의 폐기(30.2%), 장례식장·화장장·수목장 유치(27.6%) 등도 필요한 것으로 조사됐다.

황규성 한국엠바밍 대표는 “평균 수명이 짧은 동물의 죽음은 더 자주 발생하지만 전국의 합법적 장례 업체는 60여 곳이 전부다. 사람의 장례가 3일상을 치르며 상실과 슬픔을 처리할 물리적 시공간을 갖는 것처럼, 앞으로는 반려인들이 죽음을 준비할 수 있는 적절한 장례시설과 추모 공간이 확대되어야 할 것”이라고 말했다.
생명이 있는 존재의 죽음은 필연적인 일이다. 한국웰다잉문화운동 원혜영 대표는 “사람과 더불어 사는 동물의 웰다잉을 위해서는 보호소 입양 때부터 장례에 대한 교육을 더해 죽음을 준비할 수 있도록 안내해야 한다”고 말했다.

**장례 업체 이용 만족도**

한편 응답자 가운데 실제로 장례 업체를 이용해 본 사람은 적었지만, 만족도는 높은 편이었다. 가정 혹은 일상 생활에서 반려동물의 죽음을 경험하고 장례를 치러본 응답자는 39.1%(536명)로, 이 중 27.4%(147명)만 장례 업체를 이용했다.
장례 업체 서비스에 대한 응답 비율은 만족이 61.2%, 보통 28.6%, 불만족은 10.2%였다. ‘어느 정도 만족했다’(50.3%)와 ‘매우 만족했다’(10.9%)고 답한 응답자들이 서비스를 좋게 평가한 이유로 적은 중복된 의견은 ‘시설과 서비스가 깔끔했다’ ‘장례 절차가 만족스러웠다’ ‘예의와 정성을 갖추는 모습이 좋았다’ 등이었다.
반면 불만족한 점으로는 ‘거리가 멀고 주변에 없어 아쉬웠다’ ‘이용 가격이 비싼 편이다’ ‘애도 시간이 충분하지 않았다’는 것이었다.

**펫로스 증후군**

펫로스 증후군은 반려동물을 잃고 겪는 우울감, 죄책감, 수면장애, 식욕부진 등의 증상이 복합적으로 나타나거나 오랜 기간 이어져 일상생활의 어려움을 느끼는 것을 일컫는다.

‘반려동물과 이별한 사람을 위한 책’의 저자 이학범 수의사는 “반려동물과 이별하며 슬픔을 겪게 되는 것은 자연스러운 일이지만, 여러 증상이 함께 나타나거나 기간이 길어진다면 펫로스 증후군을 의심하고 전문가의 도움을 받아야 한다”고 조언한다.

**사례1**

이현정씨(60대, 서울 서초구)가 그랬다. 이씨에게 반려견 ‘페넬로피’는 “손주보다 예쁘고 자식보다 애틋한 존재”였다. 2남1녀가 장성해 각자의 가정을 꾸려 나가자, 페넬로피는 이씨에겐 마지막 품 안의 자식이 됐다. 페넬로피는 9살이 되던 해 심장병이 발병했다. 약만 잘 먹이면 15살까지는 살 거라는 생각으로 2년간 동물병원을 돌며 간병을 시작했다.
“마지막 6개월은 수액이라도 맞추려고 매일 동물병원에 갔어요. 하루만 더 살아줘, 하루만 더 살아줘 애원했죠. 마지막에 심폐소생술을 3번이나 했다고 해요. 죽고 나니까 그것도 제가 고통스럽게 한 것 같아서 모든 게 후회스럽더라고요.” 2021년 3월, 페넬로피가 떠나자 그는 불면증과 무기력증에 시달리다 같이 죽고 싶다는 생각까지 들었다.

**사례2**

정은영씨(40대, 서울 서대문구)는 반려묘 ‘삼동이’의 죽음을 황망한 이별이라고 했다. 심장이 약한 것은 알았지만 출장을 떠난 사흘 사이 고양이가 떠날 거라고는 예상치 못했다. 9년을 매일같이 털 부비며 지내던 반려묘가 세상에서 사라진 지 1년 반, 그는 “아직도 말을 꺼낼 때마다 너무 힘들다”고 했다.

**사례3**

‘아이를 보내고 나니 청춘의 한 페이지가 뜯겨져 나간 것 같다. 뜯긴 자리가 너덜너덜하고 공허하다. 밥 먹다 목이 메어 밥을 물리고, 이를 닦다가 울고, 물을 마시다 울고, 앉아서 울고, 일어나서 울고, 누워서 울고, 베개가 마를 날이 없다.’
반려인 차윤주씨는 2019년 15살 고양이 ‘미미’를 잃고 1년 이상 상실감과 죄책감, 무기력, 우울감 등에 휩싸였다. 활동적인 성격이었지만 괜찮은 척 하기 싫어 사람 만나는 걸 기피하게 됐다. 사회 활동도 줄어들었다. 반려동물을 잃고 복합적인 슬픔이 지속되는 증상인 펫로스 증후군(Pet loss Syndrome)을 경험한 것이다.

![](https://flexible.img.hani.co.kr/flexible/normal/960/721/imgdb/original/2022/0118/20220118500813.jpg)

지난 2019년 고양이 ‘미미’를 잃은 반려인 차윤주씨는 기일이 되면 평소 미미가 생전에 좋아하던 음식을 놓고 고양이를 추모하는 시간을 갖는다. 차윤주씨 제공

**펫로스 증후군 극복 시간**

애니멀피플이 공공의창·한국엠바밍·웰다잉문화운동과 함께 진행한 ‘한국 반려동물 장례 인식조사’(여론조사 기관 조원씨앤아이)에 따르면, 펫로스 증후군을 경험한 응답자 267명이 복합적인 슬픔에서 벗어나 일상을 회복하는 데 걸린 시간은 평균 2년(732.2일)이었다.

**증후군의 주요 원인과 펫로스 증후군 인식 정도**

죄책감(52.8%)과 우울증(19.5%), 반려동물의 죽음 자체에 대한 부정(18.7%) 등이었다. 반려동물의 죽음을 깊이 아파하는 사람이 있는 한편, 응답자 1천명 중 절반 이상은 ‘펫로스 증후군에 대해 오늘 이전에 들어본 적이 없다’(53.9%)고 답했다.

**인정받지 못하는 슬픔?**

실제로 보호자에게 반려동물의 죽음은 ‘가족’을 잃은 슬픔이지만 사회적으로는 종종 감춰야 할 개인사정이 됐다. 2019년 6월 만성신부전을 앓던 고양이 ‘미미’를 잃은 차윤주씨(30대, 충남 아산시)는 반려묘를 보낸 뒤 삶이 뿌리부터 흔들릴 정도로 상실감을 느꼈다. 그러나 일상적인 관계를 맺는 사람에겐 감정을 털어놓기 어려웠다. “사회생활 잘하는 사람인 줄 알았더니 고양이에 미친 여자”라는 비난을 들을까 두려웠다.

2000년대 초반 반려인구가 입양한 ‘1세대 동물들’이 속속 떠나고, 20년간 반려문화가 많이 자리잡았지만 펫로스에 대한 인식은 여전히 ‘인정 받지 못하는 슬픔’(disenfranchised grief, 밀리 코다로 ‘[반려동물 상실과 박탈된 슬픔’, 2012](https://www.researchgate.net/publication/277910130_Pet_Loss_and_Disenfranchised_Grief_Implications_for_Mental_Health_Counseling_Practice))으로 남아있는 것이다.

**펫로스 증후군의 심각성**

미국수의사회(AVMA)는 반려동물이 죽었을 때 느끼는 슬픔을 가장 가까운 친구, 가족 혹은 자녀를 잃은 슬픔과 비슷하다고 설명한다. 때로는 반려동물의 죽음이 사람보다 더 큰 상실을 준다는 연구도 있다.(월러스 사이프 ‘반려동물의 상실’ The Loss of Pet, 1993) 미국은 때문에 1990년대부터 펫로스와 관련한 여러 사회적 서비스가 운영되고 있다. 반려동물협회에서 24시간 상담 전화를 제공하거나 펫로스를 겪는 반려인들을 위한 모임을 개최하고, 여러 수의과대학에서 펫로스 지원 프로그램을 마련해 수의대 학생들이 직접 상담사로 나서기도 한다.

**펫로스 증후군 경험 수준**

애니멀피플이 공공의창·한국엠바밍·웰다잉문화운동과 함께 ‘한국 반려동물 장례 인식조사’(2021년 12월21일~23일, 표본오차는 95%, 신뢰수준에서 ±3.1%포인트)를 벌인 결과, 반려동물의 죽음을 지켜본 반려인 49.8%가 펫로스 증후군을 경험한 것으로 나타났다.

국민 네 명 중 한 명이 반려인(‘2020년 동물보호에 대한 국민의식조사’, 농림축산식품부)인 가운데 반려동물의 죽음을 경험한 보호자 중 절반이 펫로스 증후군을 겪었다는 조사 결과가 나왔다.

펫로스를 경험한 응답자의 49.8%(267명)가 가장 힘들었던 점으로 꼽은 것은 ‘잘 돌보지 못했다는 죄책감’(52.8%)이었다. 우울증(19.5%), 반려동물 죽음 자체에 대한 부정(18.7%), 죽음에 대한 분노(7.9%) 등이 뒤를 이었다. 일상을 회복하는 데 걸린 시간은 1~31일이 5.7%, 32~100일 12.8%, 101~180일 12.5%, 181~365일 18.1%, 366~730일 25.7% 등으로 나타났다. 731일 이상이 걸렸다는 답변도 25.3%나 됐다.

응답자 1천 명 중 반려동물과 생활한 경험이 있다고 답한 비율은 70.3%였으며, 이들 가운데 76.2%(536명)가 동물을 죽음을 경험한 적이 있다고 답했다. 이번 조사는 여론조사 전문기관 조원씨앤아이가 전국 18살 이상 성인남녀 1천명을 대상으로 온라인으로 설문했다.

**무지개 다리를 건너기 직전 증상**

동물들은 세상을 떠나기 직전 어떤 행동과 증상을 보일까. 이학범 수의사는 다음의 몇 가지 징후들이 나타난다면 마음의 준비가 필요하다고 조언한다.

먼저 신체적 변화로 알 수 있다. 식욕이 없어지고, 체중이 감소한다. 음식과 물을 거부하는 것은 죽기 직전 보이는 행동 변화 중 하나다. 강제 급여를 해볼 수 있겠지만 마지막에는 입에 넣어준 음식조차 삼키지 못한다.

대소변을 못가리는 것도 떠나기 직전 보이는 증상이다. 누워있는 채로 대소변을 보게 되면 물티슈로 닦아주고, 최대한 자세를 자주 바꿔주는 게 좋다. 최후의 순간에는 체온이 빠르게 떨어진다. 배나 다리를 만져봤는데 몸이 차가워졌다면, 이별의 순간이 아주 가까웠다고 생각해야 한다. 담요를 깔거나 난로를 틀어 마지막이 춥지 않도록 배려해주는 것이 좋다.

특정한 행동이 마지막을 암시하기도 한다. 가장 흔하게 보이는 모습은 어디론가 숨으려는 행동이다. 어둡고 조용한 곳을 찾아 들어가는 경우가 많은데, 이유는 여러 가지로 해석될 수 있다. 야생의 본능이 남아있기 때문일수도 있고, 사랑하는 가족에게 마지막을 보여주고 싶지 않아서 하는 행동일 수도 있다. 또 다양한 외부자극을 피해 조용하고 어두운 곳에서 안정감을 얻으려는 것일 수 있다.

만약 나이든 동물이 계속 조용한 곳으로 숨으려 한다면, 억지로 밝은 곳으로 꺼내려 하지 말고 스스로 죽음을 준비할 수 있도록 주변 환경을 정돈해주는 것이 좋다. 가만히 옆에 앉아 동물을 쓰다듬거나 낮은 목소리로 행복한 순간을 들려주고, 평소 좋아했던 장난감이나 물건을 가져다주는 것도 도움이 된다.

이학범 수의사는 다만 평가표는 “우리나라 환경과 다를 수 있고, 보호자의 심리 상태에 따라 결과가 달라질 수 있는 주관적 평가표라는 점을 고려해 수의사의 조언을 참고하는 것이 좋다”고 적고 있다.

집에 간직할 수 있는 추모물품을 만들어 둘 수도 있다. 점토로 발도장을 만들어두거나 털을 잘라 유리병 안에 보관할 수 있다. 장례에 대비해 평소 장례 때 입히고 싶었던 수의를 미리 짓거나 장례 사진을 골라보는 것도 좋다. 장

례 뒤에는 골분을 이용해 ‘메모리얼 스톤’ 등을 제작하는 방법도 있다.

펫로스의 감정도 미리 생각해보는 것이 좋다. 힘든 시기를 이겨내는데 도움이 될만한 책과 정보, 자료를 찾아보고 마음의 준비를 해두는 것이다. 언제든 도움을 구할 수 있는 펫로스 상담 프로그램들을 검색해 놓는 것도 도움이 된다. 반려동물을 보내고 너무 힘들 것 같다면 둘째의 입양을 고려해 볼 수도 있다. 첫째가 떠나고 ‘대체품’ 찾듯 둘째를 입양할 것이 아니라, 어린 둘째를 들여 미리 가족을 죽음에 대비하는 것이다.

**경조휴가**

애니멀피플이 공공의창·한국엠바밍·웰다잉문화운동과 함께 기획하고 여론조사회사 조원씨앤아이가 진행한 ‘한국 반려동물 장례인식조사’(2021년 12월 21~23일, 성인 1000명 대상)에 따르면, 아직까지 동물의 장례 휴가나 조의금 전달은 큰 공감을 얻지 못하고 있는 것으로 나타났다.

응답자들은 ‘반려동물 사망 때 직장이나 소속 단체는 휴가를 보장해야 할까’라는 설문에 높은 비율로 공감하지 못한다(76.3%)고 답했다. 휴가를 보장해야 한다는 의견(23.7%)에 비해 3배나 높은 응답율을 보인 것이다. 조의금(위로금)을 주는 문화에 대해서도 응답자의 86.2%는 공감하지 못한다고 답했다.

이들이 적정한 휴가일로 꼽은 날은 평균 2.1일이었다. ‘직장이 유급 휴가를 보장한다면 며칠이 적당한가’를 묻는 질문에 응답자의 절반(56.9%)은 하루라고 답했다. 1일부터 15일까지 자유롭게 기입할 수 있도록 한 설문에서 1일은 56.9%, 2일이 17.5%, 3일이 15.2%, 4~7일이 7.0%, 8일 이상이 2.4%로 날짜가 늘어날수록 응답율은 낮아졌다. 전체 응답자의 70.3%가 반려동물을 키운 경험이 있는 것을 감안하면 유급휴가나 조의금 복지에 대한 공감도는 높지 않았던 것이다.

지난 2017년 10월 이탈리아에서는 흥미로운 재판이 진행됐다. 로마 사피엔차 대학의 교직원이 [**반려견의 긴급 수술을 앞두고, 이틀간의 유급 휴가를 신청**](https://www.bbc.com/news/world-europe-41594224)
했지만 그의 상관은 휴가를 허락하지 않았다. 교직원은 이에 불복해 소송을 냈고, 결국 승소했다. 법원은 반려견의 수술과 병 간호가 “가족 또는 개인과 관련한 심각한 사유”에 해당한다며 급여 삭감 없이 휴가를 줘야 한다고 했다. 이 판결은 반려동물도 가족임을 제도적으로 인정한 첫 사례로 평가받는다.

2020년 반려견 ‘복돌이’를 떠나보낸 직장인 변규홍씨는 업무 중 가족으로부터 사망 소식을 전달받았다. 직장에서 바로 출발해 장례식장에 도착했을 때는 저녁 7시가 넘은 시간이었고, 장례를 마치고 유해(메모리얼 스톤)을 들고 집으로 돌아왔을 땐 이미 밤 11시가 넘은 시각이 됐다.
급히 처리할 업무 탓에 다음날 바로 출근을 해야 할 상황이었지만 마음을 추스르기 어려웠다. 변씨는 “할머니 장례 때 심적으로 굉장히 힘들었는데 주말을 제외하고도 이틀의 연차를 썼던 기억이 난다. 복돌이 때도 회사에 제도가 있었다면 바로 쉬었을 것 같다”고 말했다.

**반려동물 시장**

지난해 KB금융그룹이 발간한 '2021 한국반려동물보고서'에 따르면 집에서 동물과 함께 생활하는 인구가 무려 1448만명에 달한다. 우리나라 총 인구의 29.7%에 달하는 수치다. 함께 사는 동물을 지칭하는 표현도 '귀여워하며 즐긴다'는 뜻이 담긴 '애완동물'에서 '더불어 함께 한다'는 의미가 담긴 '반려동물'로 서서히 변화하는 추세다.

강남구가 반려동물의 죽음으로 고통받는 구민들을 위해 전문 상담사와 함께 하는 프로그램 '마음치유모임 with 펫로스' 참여자를 8월 11일부터 19일까지 모집한다.

![](https://www.gangnam.go.kr/upload/editor/2022/07/11/7d9206e4-4c8d-4c83-90f4-72961e175ceb.jpg)

펫로스 증후군을 극복하고 싶은 구민이면 누구나 신청할 수 있다. 상담은 8월 24일부터 9월 14일까지 어라운디 심리상담센터(봉은사로44길48)에서 4주간 진행한다. 이번 시범운영에는 시간대를 2클래스로 나눠 진행하며 1클래스당 최대 8명까지 모집한다. 전문가의 심리분석에 이어 반려동물과의 기억을 정리하고 죄책감이나 우울감을 건강하게 극복하기 위한 상담을 진행한다.상담에 참여하고 싶은 구민은 신청기간 내 강남구청 홈페이지에서 신청하면 된다. 선착순으로 4회 상담에 모두 참여할 수 있는 구민을 우선 선정한다. 구는 늘어나는 반려동물 인구의 행복한 삶을 위해 반려견 순회놀이터를 운영하는 한편, 제3회 반려동물 사진공모전을 8월에 실시할 예정이다.

# **정신적 증상**

펫로스 증후군의 정신적 증상으로는 다음과 같은 것들을 들 수 있습니다.

- 심한 우울감

반려동물의 죽음뿐만 아니라 모든 사별은 깊은 슬픔을 초래할 수 있습니다. 함께 했던 개나 고양이에 대한 사랑은 지극합니다. 그들의 죽음이라는 고통스러운 상황은 크나큰 슬픔입니다.

- 집중 불가

반려동물이 죽으면 어떤 일을 하면서도 개나 고양이 또는 다른 반려동물 등에 대해 끊임없이 생각을 하게 되고 온통 그 생각에 팔리게 될 수도 있습니다. 그 고통스러운 생각 때문에 일이 손에 잡히지 않게 됩니다.

- 무기력

생각할 수 있는 것이라곤 오직 세상을 떠난 개나 고양이에 대한 것뿐이기 때문에 모든 일에 흥미를 잃게 됩니다. 그에 따라 어떤 기쁨이나 즐거움도 느끼기 어렵게 됩니다.

- 공황

반려동물의 죽음이 정신적 공황(착란)을 야기할 수 있습니다.

- 죄의식

생전에 잘해 주지 못한 것을(잘해 주지 못했다고) 슬퍼할 뿐만 아니라 죄의식을 느끼기도 합니다.

# **신체적 증상**

[펫로스 증후군의 신체적 증상](http://www.psychiatricnews.net/news/articleView.html?idxno=30146)은 다음과 같습니다.

- 갑자기 슬퍼지며 울음을 멈출 수가 없습니다.

반려동물의 죽음으로 인하여 지극히 슬퍼하고 있는 동안에는 개나 고양이를 보게 된다든가 사별한 반려동물을 생각나게 하는 상황에 접하게 된다든가 하는 등 극히 사소한 자극에도 고통스러운 감정이 북받쳐 올라 울음을 멈출 수 없게 되기도 합니다.

- 식욕 상실 / 과식

슬프거나 힘든 시기를 겪게 되면 활력이 떨어지고 식욕을 잃게 됩니다. 그런가 하면 음식을 먹어서 당면한 상황의 고통을 완화시키려고 하는 사람들도 있습니다.

- 설사 / 변비 / 욕지기 / 두통 / 발열

사랑하는 반려동물의 죽음은 스트레스의 주요 원인이 되기도 합니다. 이 스트레스가 신체에 압박을 주어 통증을 야기할 수 있습니다.

- 현훈 / [청력 상실](https://www.msdmanuals.com/ko/%ED%99%88/%EC%9D%B4%EB%B9%84%EC%9D%B8%ED%9B%84%EA%B3%BC-%EC%9E%A5%EC%95%A0/%EC%B2%AD%EB%A0%A5-%EC%83%81%EC%8B%A4-%EB%B0%8F-%EB%82%9C%EC%B2%AD/%EC%B2%AD%EB%A0%A5-%EC%83%81%EC%8B%A4)/ 환각

고통스러운 정신 상태는 여러 가지로 신체에 영향을 미쳐 신체가 정상적으로 기능할 수 없게 하기도 합니다.

- 불면증

일상생활 중 어려운 시기를 겪게 되면 수면에 장애가 올 수 있습니다. 똑같은 증상인 수면 장애가 펫로스의 경우에도 나타날 수 있습니다.

# 반려동물의 죽음에 따른 비탄의 5단계

반려동물의 [죽음에 따른 비탄의 단계](https://m.blog.naver.com/PostView.naver?isHttpsRedirect=true&blogId=jinz91&logNo=80176220467)는 다음과 같습니다.

1. 부정

이 단계에서는 반려동물의 죽음을 받아들이지 못합니다. 너무나도 충격적인 반려동물의 죽음 앞에서 그야말로 현실적으로 벌어진 그 고통스러운 상황을 감내할 수가 없습니다

1. 분노

이 단계에서는 「그때 이러저러하게 했어야 하는 건데」 하는 감정이 생기게 되고 가족과 수의사 그리고 자기 자신을 탓하게도 됩니다.

1. 흥정

이 단계에서는 어떻게 한들 죽은 반려동물이 살아날 수 없다는 현실에 직면해서 반려동물을 살릴 수만 있다면 자기 목숨이라도 내놓겠다는 생각까지도 하게 됩니다.

1. 우울증

죽은 반려동물이 살아 돌아올 수 없다는 것을 깨닫게 되면서 고통스러운 비탄에 빠지게 됩니다. 이 단계에서는 펫로스(증후군)에 따른 여러 가지 증상을 겪게 됩니다.

1. 수용

이 단계에서는 펫로스의 슬픔으로부터 회복되어 본정신으로 돌아옵니다. 한때는 개나 고양이 또는 다른 반려동물 밖에 생각할 수 없었던 사람들이나 여러 가지 정신적 신체적 증상을 겪었던 사람들이 펫로스 증후군의 증세가 약화되거나 사라진 것을 지각하게 됩니다.

나아가 이 단계에서는 펫로스 증후군의 증상이 사라질 뿐만 아니라 (새로운) 반려동물에 대한 사랑의 감정도 돌아오게 됩니다. 죽은 반려동물과의 추억도 잊어버리지 않으면서 말입니다.펫로스 증후군의 증상이 사라지면 신체적 정신적 건강도 회복됩니다. 따라서 죽은 반려동물의 화장 다이아몬드를 만들어 추억을 가까이에서 간직하는 것과 같은 해결책을 찾는 것도 좋은 아이디어라고 할 수 있겠습니다.

**극복법**

- 감정 표현하기

눈물은 펫로스에 따른 감정 표현의 일부입니다. 하염없이 울고 있는 것은 좋지 않지만 눈물을 억지로 참는다든가 고통스러운 감정을 억누르는 것은 오히려 스트레스를 유발하여 펫로스를 극복하는 데 장애가 됩니다. 울고싶을 때에는 울어서 감정을 표출하도록 하십시오.

- 편안한 사람과 대화하기

펫로스는 받아들이기 어려운 일일 수 있습니다. 그러므로 펫로스의 경험이 있는 사람들이나 사별한 자기의 반려동물과 친했던 가족 또는 친구들과 추억담을 나누는 것이 펫로스 극복 과정을 단축시키는 하나의 방법이 됩니다.

- 좋아하는 일에 몰두하기

무언가 다른 일에 몰두하는 것도 고통스러운 생각으로부터 벗어나는 하나의 해결책입니다. 그렇게 하는 것이 펫로스를 극복하는 아주 좋은 방법입니다. 보다 많은 시간을 들여 좋아하는 일에 몰두해 보십시오.

- 장례를 치르고 작별하기

반려동물의 죽음은 크나큰 고통을 안겨 주지만 적절한 뒷수습은 펫로스를 극복하는 데 도움이 될 수 있습니다. 펫로스를 극복하는 한 가지 방법으로 비용이 더 많이 들긴 합니다만 죽은 사람의 경우처럼 [슬픔 케어의 일부로서 반려동물 장례를 치러 주는 것](https://blog.naver.com/hanlaonline/221735148011)을 들 수 있습니다. 이렇게 하면 사랑하는 개나 고양이의 죽음을 받아들이고 펫로스에 따른 여러 가지 증후의 발현을 예방하는 데 도움이 될 것입니다.

- 무덤이나 제단 만들기

무덤이나 제단을 만드는 것도 펫로스를 극복하는 또 하나의 방법입니다. 약간의 돈이야 들겠지만 사별한 반려동물의 무덤을 방문하여 제단에 손을 얹고 추억을 떠올릴 수 있으면 마음으로나마 반려동물과 재회할 수 있게 되고 그에 따라 슬픔도 누그러집니다.

- 메모리얼 주얼리 만들기

## 실태 및 문제

### 한국 직장인의 운동부족 실태

- 한국건강증진개발원 자료에 따르면, 직장인 중 절반 이상이 하루에 걸어서 이동하는 시간이 30분도 안 되는 것으로 나타남.
- 2016년 기준, 출퇴근 및 점심시간을 포함한 하루 중 걷는 시간이 30분 미만인 비율: 52.0%
- 하루에 60분 이상 걷는 비율: 14.4%
- 체육활동 가능 시간부족, 접근성 낮음, 관심 부족, 지출비용 부담

### 청소년의 운동부족 실태

- 세계보건기구(WHO)의 2019년 조사 결과, 한국 청소년들의 운동 부족이 전 세계적으로 가장 심각한 수준으로 나타남.

     - 한국 청소년 운동부족 비율: 94.2%

     - 여자 청소년의 운동부족 비율: 97.2% (세계 최고)

     - 남자 청소년의 운동부족 비율: 91.4% (필리핀 다음으로 높음)

- 학생들은 입시위주의 한국교육
- 야외활동 줄고 원격수업 등 영향

### Part 1. 중고거래

### 1. 개요 및 목적

중고거래 플랫폼은 많은 사람들이 저렴하게 물건을 사고 팔 수 있는 유용한 공간이다. 그러나 이용자들은 여전히 몇 가지 불편한 점을 겪고 있다. 본 조사에서는 소비자들이 이용하는 중고거래의 현황과 문제점 및 해결방안을 탐색해보고자 한다.

### 2. 관련 현황

**2-1. 성장하는 중고거래(리커머스) 시장**

- 국내 중고 거래 시장은 2008년 4조원에서 2021년 24조, 오는 2025년은 43조원 규모로 급성장할 것으로 예상된다. (한국인터넷진흥원, 신한투자증권)
- 리커머스(Re-commerce)는 중고 거래나 명품ㆍ한정판 제품 재거래를 뜻하며, 전 세계 중고 거래 시장 규모는 2021년 270억 달러에서 2025년 770억 달러로 성장할 전망임.
- 온라인뿐만 아니라 백화점 등 오프라인에서도 중고품 유통 채널을 확대하고 있음.

**예시**

- 구구스: 2023년 거래액 2153억원, 전년 대비 약 20% 증가
- 롯데쇼핑과 신세계그룹: 중고 거래 플랫폼 투자 및 직접 매장 운영
- 현대백화점 신촌점 유플렉스: 중고품 전문관 '세컨드 부티크' 운영
- 브랜드나라: 2022년 서울 현대백화점 미아점에 '럭스어게인' 입점
(출처: [https://m.dnews.co.kr/m_home/view.jsp?idxno=202402041401031630530](https://m.dnews.co.kr/m_home/view.jsp?idxno=202402041401031630530))

**2-2. 3세대 중고 거래 플랫폼 등장**

- 국내 중고 거래 플랫폼 시장이 진화하고 있다. 중고 제품 중개 서비스가 많아지고, 비즈니스 모델도 다변화하는 모습이다.
- 1세대 중고 거래 플랫폼으로는 2003년 네이버의 카페 서비스로 시작한 '중고나라'가 있다. 2세대 플랫폼으로는 '당근'과 '번개장터'가 있으며, 당근의 월간활성이용자(MAU)는 1900만 명에 달한다.
- 최근에는 기존 중고 거래 플랫폼의 단점을 해결한 '3세대 중고 거래 플랫폼'이 주목받고 있다. 패션 등 특정 분야에 집중한 것이 특징이다.

**예시**

- '차란': 출시 8개월 만에 이용자 수 20만 명 돌파, 100억원 규모 시리즈A 투자 유치. 소비자의 중고 의류를 판매 대행하는 서비스 제공.
- '후루츠패밀리': 의류, 신발 등 패션 전문 중고 플랫폼. 지난해 거래액 300% 증가.
- '콜렉티브': 다양한 국내외 프리미엄·디자이너 중고 패션 아이템 거래. 오프라인 매장 운영.
- '리클': 헌 옷 수거 및 중고 의류 판매 스타트업. 시리즈A 투자 유치, 재판매율 80% 이상 유지.
- '퀵셀': 중고 전자제품 거래 서비스. AI를 통한 제품 상태 진단 및 구입.
- '페이브릴': 중고 명품 주얼리 전문 플랫폼. 전문가 검수 및 세부 정보 자동 등록.
(출처:     [AI가 가격 책정, 헌 옷 방문 수거…중고 거래 플랫폼 세대 교체 [긱스] | 한국경제 (hankyung.com)](https://www.hankyung.com/article/2024052890941)   )

### 3. 문제점 및 해결방안

**3-1. 사기문제**

- **문제점**
    - 중고거래 사기 급증: 2023년 사기 피해 건수 31만 건, 피해 금액 2600억 원 (출처: 더치트)
    - 개인 간 거래 특성상 사기 범죄 발생 빈도 높음
- **개선 방안**
    - 중고나라: 카카오톡 채널 챗봇 연동으로 사기 이력 조회 시스템 강화
    - 당근마켓: 비대면 거래 시 경고 메시지 시스템 구축, 대면 직거래 강조
    - 번개장터: 광학문자인식(OCR) 기술 도입, 사기 징후 차단 시스템
    (출처: [[기획] 중고거래 플랫폼, '사기 피해 진화' 어떻게 나서나? - 매일일보 (m-i.kr)](https://www.m-i.kr/news/articleView.html?idxno=1136737))

**3-2. 가품문제**

- **문제점**
    - 가품 거래량 증가: 업계는 모조품 판매 증가와 처벌의 어려움을 지적
    - 크림의 2024년 상반기 데이터에 따르면, 가품 건수는 전년 동기 대비 3배 증가
    - 주요 가품 출처: 온라인 패션 플랫폼 (56%), 개인 간 거래 플랫폼 (28%), 공구·직구 플랫폼 (7%)
- **개선 방안**
    - **크림 사례**
        - 전담 검수센터 운영 및 철저한 가품 검수
        - 가품 판별 가이드 제공: 높은 비율의 가품이 발견된 브랜드에 대한 판별 가이드 제공 (ex. 스투시 반팔 티셔츠 2022년 버전)

(출처 : [‘짝퉁’ 신발부터 옷까지...크림 “가품 거래 3배 늘었다” (kukinews.com)](https://www.kukinews.com/newsView/kuk202407040192))

**3-3. 대면 및 개인정보 노출 문제**

- **문제점**
    - 판매자는 주로 구매자와의 연락 및 가격 조율(51%), 거래 준비 과정(37%), 개인정보 노출(34%), 일방적인 거래 취소(29%) 등을 불편한 점으로 꼽고 있다.
    - 반면 구매자는 거래물품 품질 및 상태 확인(47%), 사기 거래에 대한 불안감(46%), 판매자와의 연락 및 가격 조율(33%), 개인정보 노출(22%) 등의 이유로 불편함을 느낀다.
    - 이를 통해 구매자와 판매자 모두 접촉을 최소화하는 것을 목표로 삼으면서도 상품 상태를 확인하고 싶어하는 것을 알 수 있다.

출처 : [컨슈머인사이트 (consumerinsight.co.kr)](https://www.consumerinsight.co.kr/voc_view.aspx?no=3381&id=ins02_list&PageNo=1&schFlag=0)

- **개선 방안**
    - **파라바라 앱**:
        - 판매자는 판매하고자 하는 물품을 앱에 올리고, 다른 이용자들로부터 하트 3개 이상을 받으면 가까운 파라박스에 물건을 넣고 판매할 수 있다.
        - 구매 희망자는 물건이 비치된 매장을 찾아가 직접 눈으로 확인하고, 마음에 들면 신용카드나 체크카드로 결제하고 가져갈 수 있다.

### 4. 서비스 개발 방향

해커톤 때 논의했던, 주제에서 파라바라의 중고거래 박스를 응용해, 비대면 안전 거래가 가능하도록 진보시킨다.

### Part 2. 관광 지역축제 한국문화 문화재

### **1. 조사 개요 및 목적**

- 관광 지역축제와 한국문화, 문화재는 국내외 관광객들에게 전통과 문화, 지역 특산품 등을 소개하는 중요한 역할을 한다.
- 본 조사는 이러한 지역축제의 현황과 문제점 및 개선 방안을 탐색하고자 한다.

### **2. 관련 현황**

- 지역축제는 지역 경제 활성화에 큰 도움이 되는 동시에 지역의 문화와 특산품을 홍보하는 중요한 수단이다.

출처:

[강진군, 축제 통한 지역경제 밑불 효과 ‘입증’ | Save Internet 뉴데일리 (newdaily.co.kr)](https://gj.newdaily.co.kr/site/data/html/2024/07/05/2024070500167.html)

[광주 충장축제 기간 지역 상권 매출 41％ 늘어 | 연합뉴스 (yna.co.kr)](https://www.yna.co.kr/view/AKR20240704100500054?input=1195m)

### 3. 문제점

- **바가지 요금**: 지역축제에서 지나치게 높은 가격의 음식과 상품이 판매되는 경우가 많음
- 기타 : 상징성 부족, 안전 대책 미흡 등도 존재.
- **개선 방안**
    - **가격 표기 의무화**: 모든 판매자는 가격을 명확하게 표기하고, 이를 준수하도록 한다.
    - **집중 단속**: 축제 기간 동안 바가지 요금 근절을 위한 집중 단속을 실시하고, 위반 업체에 대한 강력한 처벌을 시행한다.
    - **자정 작용 유도**: 지역 상인들이 자발적으로 가격을 조정하고, 합리적인 가격을 유지하도록 유도한다.
    - **민원 접수와 처리**: 축제 기간 동안 바가지 요금 관련 민원을 신속히 접수하고 처리하는 시스템을 운영한다.

(출처 : [어묵 한 그릇이 만원?…지역축제 바가지에 놀란 日 유튜버 - 아시아경제 (asiae.co.kr)](https://view.asiae.co.kr/article/2023053013584739717))
